# node_todo
Simple nest api to handle two entities. 
- user
- blog
## How to install

to install you need postgresql, node and npm installed on your machine.

clone the the project and change direcotry and install dependencies
```
git clone
cd blog-management-system
npm install 
npm run build 
npm run start:prod.
```
